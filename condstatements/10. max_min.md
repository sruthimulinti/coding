## max number and min number
```c
#include<stdio.h>
main()
{
        int a,b;
        scanf("%d %d",&a,&b);
        if(a>b)
        {
            printf("a is max");
            printf("b is min");
        }
        else
        {
            printf("a is min");
            printf("b is max");
        }
}
```
